# logo/

Brand & icon assets for **loopy**.

- **`loopy-brand/`** — the official, organized asset set. Start with
  `loopy-brand/README.md` and `loopy-brand/PREVIEW.png`.
  Contains: `svg/`, `png/`, `favicon/`, `macos/`, `windows/`, `src/` (generator),
  and `source/` (original concept reference).
- **`_archive/`** — a previous export kept for reference (loose PNGs/SVGs and
  generator scripts). Safe to delete once you're happy with `loopy-brand/`.

Everything is generated from a single clean vector master — see
`loopy-brand/src/` to tweak colors/geometry and re-run `python3 build.py`.
